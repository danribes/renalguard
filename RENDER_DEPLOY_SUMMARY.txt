üÜò TROUBLESHOOTING QUICK REFERENCE
================================================================================

Problem: Build fails
Solution: Check Dockerfile and package.json in respective directories

Problem: Database connection fails
Solution: Use Internal Database URL (not External) in backend env

Problem: CORS errors in browser
Solution: Update CORS_ORIGIN in backend to match frontend URL

Problem: AI analysis fails
Solution: Verify ANTHROPIC_API_KEY is set correctly in backend

Problem: Slow first load (30-60s)
Cause: Free tier sleep - backend waking up
Solution: Pre-warm with curl or upgrade to paid tier

Problem: Frontend can't reach backend
Solution: Check VITE_API_URL in frontend matches backend URL
